# Voice Decomposer

Разложение голоса на частоты и тона в реальном времени.

![Voice Decomposer Demo](https://via.placeholder.com/800x400/0a0a0a/00ff88?text=Voice+Decomposer)

## Функции

- 📊 Визуализация частотного спектра в реальном времени
- 🎨 Спектрограмма с историей (140 кадров)
- 🎤 Работает в браузере — никаких серверов

## Как запустить локально

```bash
# Через Python
python -m http.server 8000

# Или через Node.js (если установлен)
npx serve .
```

Открой <http://localhost:8000>

## Деплой на GitHub Pages

1. Создай новый репозиторий на GitHub
2. Загрузи все файлы (`index.html`, `style.css`, `script.js`) в корень
3. В настройках репозитория → **Settings → Pages** → **Source**: выбери `main` ветку, папку `/ (root)`
4. Через 1-2 минуты приложение доступно по адресу: `https://твойюзер.github.io/название-репо/`

**Важно:** На GitHub Pages работает через HTTPS — микрофон будет запрашиваться автоматически.
